* 删除指定字符"abc"前的所有内容
```
:%s/^.*\(abc.*\)/\1/g
```

* 删除所有空行
```
:g/^\s*$/d
```

* 删除行首指定数量的字符
```
:%s/^.\{9\}//g
```

* 替换
```
:%s/vivian/sky/（等同于 :g/vivian/s//sky/） 替换每一行的第一个 vivian 为 sky
:%s/vivian/sky/g（等同于 :g/vivian/s//sky/g） 替换每一行中所有 vivian 为 sky
```
